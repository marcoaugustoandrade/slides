## Criação do usuário no Ubuntu 

	sudo useradd nome
	sudo passwd nome
	sudo usermod -aG sudo nome

Instalação do Node com NVM


	curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash
	source ~/.bashrc
	nvm list-remote
	nvm install v20.0.0

Criação do primeiro projeto TypeScript


	mkdir aula-1-ts
	cd aula-1-ts
	code .
	npm init -y
	npm install ts-node
	npm install -g nodemon
